Simple Cache Lookup Visualizer

A small teaching demo showing, step-by-step, how an input selects an array entry and how that entry maps to a cache line.

Controls

Start with direct mode to show basic indexing, then switch to secret-mixed mode to show how a secret changes the accessed entry and cache line.
Step 1: Choose an inputThe public input is the value sent to the victim.
Step 2: Compute array indexIn direct mode, the input is used directly. In secret-mixed mode, the secret changes the index.
Step 3: Access array entryThe lookup reads one array element, like table[index].
Step 4: Touch a cache lineSeveral adjacent entries share one cache line, so accessing one entry tends to make that whole line hot.
